# Visual Style Guide

## Design Tone
- playful
- calm
- friendly
- modern

## Inspiration
- Duolingo
- Khan Kids
- Nintendo UI

## Typography
Headings: Fredoka
Body: Nunito

## Colors
Primary Blue: #5B8DEF
Purple: #8B5CF6
Gold: #FFD166
Cream: #FFF8EE

## UI Principles
- Rounded cards
- Large spacing
- Minimal text
- Soft shadows

## Avoid
- Dense layouts
- Tiny buttons
- Corporate dashboards
