# Chess Adventures

A playful chess learning game for kids ages 6–8.

## MVP Scope
- Learn chess pieces
- Solve puzzles
- Play beginner AI

## Stack
- Next.js
- Tailwind
- shadcn/ui
- chess.js

## Project Philosophy
- Simple over feature-rich
- Encouraging over competitive
- Reusable systems over custom screens
